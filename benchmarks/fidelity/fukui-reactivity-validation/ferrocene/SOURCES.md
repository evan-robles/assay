# Geometry source

- Molecule: ferrocene, Fe(C5H5)2
- Source: chemkit build-from-smiles (Open Babel obabel --gen3d), NOT a database.
- Reason: PubChem does not provide a reliable 3D conformer for this Fe sandwich
  organometallic. This geometry was generated by the toolkit's own builder and
  reused across the ferrocene edge-case folders.

Note: this is a force-field-built (obabel) starting geometry, not a database
record nor a structure optimized at the calculation's level of theory. For a
publication-grade ferrocene geometry, use a crystallographic source (e.g. COD)
or a QM-optimized structure.
